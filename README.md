# 🌍 Foundation Project Website

## 📌 Overview

This project is a website I am building as part of my learning journey at Migracode Bootcamp. It represents a personal foundation initiative focused on supporting underprivileged communities in Ghana.

The goal of the website is to raise awareness and promote initiatives around:

* Good health
* Better living conditions
* Sports development in rural communities
